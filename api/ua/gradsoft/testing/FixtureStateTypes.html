  17. val stateAspects : Enumeration

    Set of possible state aspects.

    Set of possible state aspects. Aspect here is some part of state: test can use or change different aspects of same state. I.e. if one test use one set of aspects, than other test, which use different aspects than first, can run after first without reloading of fixture state.

    By default all tests use one global aspect.
